# OliamBot
基于 [mirai](https://github.com/mamoe/mirai) 的qq聊天机器人，以~~R18~~图片发送为主要功能。具有部署简单~~个屁~~，有一定 [扩展性](https://github.com/KillerQueen3/OliamBot/blob/master/plugins/README.MD) 的特点。
* 根据关键词查找pixiv上相关图片
* 以pixiv图片id找图
* 以pixiv用户id发送用户随机作品
* pixiv推荐图片
* [随机色图](https://api.lolicon.app/#/setu)
* 弱插件支持
